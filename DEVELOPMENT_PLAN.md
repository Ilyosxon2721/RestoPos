# RestoPOS - Полный план разработки

## 📋 Обзор проекта

**RestoPOS** — облачная POS-система для автоматизации ресторанов и кафе.

### Технологический стек
- **Backend:** Laravel 12, PHP 8.3+
- **Frontend:** Livewire 3, Alpine.js, Tailwind CSS, Flux UI
- **Database:** MySQL 8.0
- **Cache/Queue:** Redis
- **Auth:** Laravel Sanctum

---

## 🏗️ Архитектура модулей

```
┌─────────────────────────────────────────────────────────────┐
│                        RestoPOS                              │
├─────────────────────────────────────────────────────────────┤
│  CORE MODULES                                                │
│  ├── Auth         - Аутентификация, роли, права             │
│  ├── Organization - Мультитенантность, филиалы              │
│  └── Settings     - Настройки системы                       │
├─────────────────────────────────────────────────────────────┤
│  BUSINESS MODULES                                            │
│  ├── Menu         - Категории, блюда, модификаторы          │
│  ├── Orders       - Заказы, столы, чеки                     │
│  ├── Kitchen      - KDS, очередь готовки                    │
│  ├── Warehouse    - Склад, поставки, инвентаризация         │
│  ├── Finance      - Кассы, платежи, P&L                     │
│  ├── Customers    - CRM, лояльность, бонусы                 │
│  ├── Staff        - Персонал, смены, зарплаты               │
│  ├── Delivery     - Доставка, курьеры, зоны                 │
│  └── Reports      - Аналитика, отчёты, дашборды             │
├─────────────────────────────────────────────────────────────┤
│  INTEGRATIONS                                                │
│  ├── Fiscal       - Фискализация (Узбекистан)               │
│  ├── Payments     - Payme, Click, карты                     │
│  ├── SMS          - PlayMobile, Eskiz                       │
│  └── Telegram     - Уведомления                             │
└─────────────────────────────────────────────────────────────┘
```

## 🚀 Быстрый старт

### Использование с Claude Code

1. Откройте проект в VS Code с Claude Code
2. Выполняйте этапы последовательно:

```
/etap-1-1
/etap-1-2
/etap-1-3
...
```

### Использование вручную

Следуйте инструкциям в папке `phases/`:
- `phases/1-foundation/`
- `phases/2-core/`
- `phases/3-management/`
- и т.д.
