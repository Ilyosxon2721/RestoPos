# RestoPOS - Быстрый старт

## 🚀 Как использовать этапы разработки

### Вариант 1: Claude Code (рекомендуется)

1. Откройте проект в VS Code с расширением Claude Code
2. Скопируйте папку `.claude/commands/` в корень проекта
3. Выполняйте этапы последовательно:

```
/etap-1-1    # Установка зависимостей
/etap-1-2    # Миграции базы данных
/etap-1-3    # Модуль Auth
/etap-1-4    # Модуль Organization
/etap-2-1    # Модуль Menu
/etap-2-2    # Модуль Orders
/etap-2-3    # Kitchen Display
```

### Вариант 2: Ручное выполнение

Откройте файлы в `.claude/commands/` и выполняйте инструкции вручную.

## ⚡ Минимальный MVP

Для запуска минимально работающей системы нужны этапы:

```bash
# 1. Установка
/etap-1-1

# 2. База данных  
/etap-1-2

# 3. Аутентификация
/etap-1-3

# 4. Организации
/etap-1-4

# 5. Меню
/etap-2-1

# 6. Заказы и POS
/etap-2-2
```

**Результат:** Работающий POS-терминал с возможностью создания заказов.

## 🔧 Полезные команды

```bash
# Запуск проекта
php artisan serve --port=8001

# Пересоздать БД с тестовыми данными
php artisan migrate:fresh --seed

# Очистить кэш
php artisan optimize:clear

# Запуск тестов
php artisan test

# Сборка фронтенда
npm run build
```
